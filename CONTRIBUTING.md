# Contributing

This repository is primarily a portfolio case study, but focused improvements are welcome.

## Good contributions

- clearer enterprise-AI governance language;
- stronger adoption or change-management templates;
- accessibility and documentation improvements;
- additional non-proprietary examples;
- corrections to risk, measurement or operating-model content.

## Out of scope

Do not submit:

- proprietary employer information;
- credentials or personal data;
- copyrighted books, papers or paid-course material;
- market data or trading strategies;
- claims of guaranteed productivity, profit or model accuracy;
- code that accesses live systems or protected data.

## Pull-request expectations

A pull request should state:

1. the problem being addressed;
2. the proposed change;
3. affected files;
4. validation performed;
5. privacy, security and licensing considerations.

Keep changes small, reviewable and reversible. Human review remains required before merge.
